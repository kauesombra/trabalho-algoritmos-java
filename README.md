# Trabalho de Computação Paralela e algoritimos em Java

Projeto acadêmico desenvolvido em Java com o objetivo de comparar o desempenho de algoritmos de ordenação em versões sequenciais e paralelas.

Foram implementados os algoritmos Bubble Sort, Selection Sort, Insertion Sort e Merge Sort, realizando testes com diferentes tamanhos de entrada, tipos de dados e quantidades de threads.

Os resultados são registrados em arquivos CSV e apresentados por meio de gráficos dinâmicos em Java Swing, permitindo análise comparativa entre as execuções.

# Recursos do projeto
- Implementação serial e paralela
- Testes automatizados
- Geração de CSV
- Gráficos comparativos
- Análise de desempenho
